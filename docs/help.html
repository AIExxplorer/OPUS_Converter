Instalação do FFmpeg
Importante: O OPUS Converter requer o FFmpeg para funcionar. Se você estiver vendo a mensagem "FFmpeg não está instalado ou não está no PATH do sistema", siga as instruções abaixo.

Método 1: Instalação Manual

Siga estes passos para instalar o FFmpeg manualmente:

  1. Baixe o FFmpeg:
    • Visite o site oficial do FFmpeg
    • Clique em "Windows Builds" na seção "Get packages & executable files"
    • Baixe a versão mais recente (geralmente de gyan.dev ou BtbN)
    • Escolha a versão "full" ou "essentials" (formato 7z ou zip)
  2. Extraia o arquivo baixado:
    • Use o 7-Zip ou a ferramenta de extração do Windows
    • Extraia para um local como C:\ffmpeg
  3. Adicione o FFmpeg ao PATH do sistema:
    • Clique com o botão direito em "Este PC" ou "Meu Computador" e selecione "Propriedades"
    • Clique em "Configurações avançadas do sistema"
    • Clique em "Variáveis de ambiente"
    • Em "Variáveis do sistema", encontre e selecione "Path", depois clique em "Editar"
    • Clique em "Novo" e adicione o caminho para a pasta bin do FFmpeg (ex: C:\ffmpeg\bin)
    • Clique em "OK" em todas as janelas para salvar as alterações
  4. Verifique a instalação:
    • Abra um novo Prompt de Comando ou PowerShell
    • Digite ffmpeg -version e pressione Enter
    • Você deve ver informações sobre a versão do FFmpeg

Método 2: Usando o Chocolatey

Se você tem o Chocolatey instalado, este é o método mais fácil:

  1. Abra o PowerShell como Administrador
  2. Execute o comando: choco install ffmpeg
  3. Verifique a instalação com: ffmpeg -version

Método 3: Usando Nossos Scripts

Fornecemos scripts que podem ajudar a adicionar o FFmpeg ao PATH do sistema:

Script PowerShell

Recomendado para usuários do Windows 10/11.

  1. Baixe o Script PowerShell
  2. Clique com o botão direito no arquivo baixado
  3. Selecione "Executar com o PowerShell"
  4. Se solicitado, confirme a execução
  5. Siga as instruções na tela
Script Batch

Alternativa para todos os sistemas Windows.

  1. Baixe o Script Batch
  2. Clique com o botão direito no arquivo baixado
  3. Selecione "Executar como administrador"
  4. Siga as instruções na tela
Solução de Problemas Comuns

Este erro ocorre quando o OPUS Converter não consegue encontrar o FFmpeg no seu sistema. Mesmo que você tenha instalado o FFmpeg, ele precisa estar no PATH do sistema para ser encontrado.

Soluções:

  1. Verifique se o FFmpeg está instalado corretamente usando o comando ffmpeg -version em um novo terminal
  2. Use um dos nossos scripts para adicionar o FFmpeg ao PATH automaticamente
  3. Reinicie o computador após adicionar o FFmpeg ao PATH
  4. Verifique se o caminho para o executável do FFmpeg está correto (deve apontar para a pasta que contém ffmpeg.exe)

Este erro geralmente ocorre quando o caminho para o arquivo de entrada ou saída contém caracteres especiais ou espaços.

Soluções:

  1. Renomeie seus arquivos para usar apenas letras, números e underscores (_)
  2. Evite usar espaços nos nomes dos arquivos
  3. Mova os arquivos para um diretório com caminho simples (ex: C:\temp)

Este erro ocorre quando você tenta converter um arquivo que não está no formato OPUS ou está corrompido.

Soluções:

  1. Verifique se o arquivo tem a extensão .opus
  2. Confirme se o arquivo é realmente um arquivo de áudio OPUS
  3. Tente obter uma nova cópia do arquivo original
  4. Use uma ferramenta como o MediaInfo para verificar o formato do arquivo
